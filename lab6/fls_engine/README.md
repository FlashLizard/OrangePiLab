# FLS_Engine

## 1. Introduction

A toy game engine written in C++.

## 2. Features

- [x] 3D rendering
- [x] base framework
- [ ] physics
- [ ] ui

## 3. Build

Download glm(>0.9.9) first

Then run

```bash
make
```

You will get libflengine.a

## 4. example

see lab6 in <https://github.com/FlashLizard/OrangePiLab.git>, 
it is a Minecraft demo using this engine.
